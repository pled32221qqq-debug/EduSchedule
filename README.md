# EduSchedule — Система управления расписанием

Информационная система для автоматизации формирования и управления расписанием занятий в образовательных организациях.

## Описание

Проект разработан в рамках курсовой работы по дисциплине «Управление и автоматизация баз данных».

**Цель проекта:** автоматизация процесса составления расписания занятий, исключение конфликтов ресурсов и упрощение управления заменами преподавателей.

## Технологический стек

- **Язык программирования:** C#
- **Фреймворк:** WPF (Windows Presentation Foundation)
- **СУБД:** MS SQL Server
- **Платформа:** .NET Framework
- **Среда разработки:** Microsoft Visual Studio

## Функциональные возможности

- Авторизация пользователей (диспетчер/методист)
- Формирование расписания занятий
- Просмотр расписания по группам и преподавателям
- Управление заменами преподавателей
- Ведение справочников (группы, преподаватели, дисциплины, аудитории)
- Автоматическая проверка конфликтов (наложение пар)
